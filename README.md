### Project Overview
X Bank is an enterprise FinTech product designed to eliminate the "Pay first - Reclaim later" culture in businesses. By leveraging **API Payout** technology, X Bank allows employees to spend directly from the company’s budget for business purposes—specifically focusing on travel (flights & hotels)—while ensuring real-time visibility and automated reconciliation for accounting teams.

---

### Core Solution: The "X Bank Travel" Module
The traditional reimbursement process takes days or weeks. X Bank Travel reduces this to minutes:
1.  **Direct Booking:** Employees book flights/hotels directly within the app using company funds.
2.  **Automated Compliance:** System automatically validates bookings against company spending policies (Business Rules).
3.  **Real-time Reconciliation:** API integration ensures that every transaction is instantly matched with its corresponding invoice/booking ID, eliminating manual work for accountants.
